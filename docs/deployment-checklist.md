# 배포 체크리스트

AWS EC2 웹 서버 구성을 만들거나 점검할 때 사용하는 체크리스트입니다.

## 인스턴스 생성 전

- [ ] 리전 선택: `ap-northeast-2`
- [ ] 적절한 AMI 선택
- [ ] 인스턴스 유형 선택: `t3.micro`
- [ ] 키 페어 생성 또는 선택
- [ ] 스토리지 설정
- [ ] 보안 그룹 규칙 설정
- [ ] 필요한 태그 추가

## 인스턴스 생성 후

- [ ] 인스턴스 상태가 `Running`인지 확인
- [ ] 상태 검사가 통과되는지 확인
- [ ] SSH 접속 확인
- [ ] 웹 서버 설치 및 실행
- [ ] 브라우저에서 웹 페이지 접속 확인

## GitHub 업로드 전

- [ ] AWS 계정 ID 제거
- [ ] 액세스 키와 시크릿 키 제거
- [ ] `.pem` 파일 제거
- [ ] 공개 저장소라면 퍼블릭 IP와 DNS 이름 제거
- [ ] 스크린샷의 민감 정보 가리기
- [ ] `.gitignore`가 비밀 정보와 로컬 파일을 제외하는지 확인
